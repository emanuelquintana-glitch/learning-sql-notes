##  PROPÓSITO DE ESTE PROYECTO

Este es un **PROYECTO EDUCATIVO PERSONAL** creado por **Emanuel Quintana Silva** 
(estudiante de Economía, UPTC) con el objetivo de:
1. Aprender y practicar SQL y programación
2. Organizar apuntes y ejemplos para estudio propio
3. Compartir conocimiento de manera educativa
4. Crear un portafolio de aprendizaje
